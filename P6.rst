P6. Повторная закупка товара/услуги/работы, по которому ранее была отменена закупка по причине отпавшей необходимости.
######################################################################################################################

Факторы, которые на данном этапе не позволяют реализовать индикатор
===================================================================

Реализацию данного индикатора делает невозможным отсутствие данных о причине отмены процедур. Даже в случае добавления этих данных будет сложно однозначно классифицировать причины отмены, так как они вводятся в произвольной форме вручную представителями тендерного комитета.

***************
Суть индикатора
***************

Аналитическая система сравнивает список кодов предметов закупки со списком кодов предметов закупок отмененных закупающей организацией в текущем календарном году.


****
Риск
****
Риск сговора между представителями закупающей организации и участником конкурса. Процедура отменяется в том случае, если невозможно отменить "неудобного" победителя конкурса.

*******************************
Нарушение норм/принципов закона
*******************************

Пункт 2 статьи 1:
"- создания равных и справедливых условий для поставщиков (подрядчиков) при осуществлении государственных закупок."

*******************************
Методология расчета индикатора.
*******************************

Уровень расчета
===============
Индикатор расcчитывается на уровне *процедуры*.

Источники данных для расчета
============================

Для расчета индикатора используются следующие источники данных:

- API системы государственных закупок в OCDS формате

Типы процедур
=============

Индикатор рассчитывается для следующих типов процедур:

- одноэтапный,
- двухэтапный,
- упрощенный,
- на понижение цены.


Статусы процедур
----------------

Индикатор расчитывается для процедур, которые:

- находятся в статусе ``complete``


Частота расчета
===============

Если выполнены все условия для активации расчета индикатор, он рассчитывается один раз.
